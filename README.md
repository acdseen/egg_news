# backend

## About

个人瞎搞的一个 egg 新闻后台项目

### 项目运行

```bash
项目运行之前，请确保系统已经安装以下应用
1、node (8.9 及以上版本)
2、mongodb (开启状态)
```

```bash
$ npm i
$ npm run dev
$ open http://localhost:7001/ （如果已启动前台程序，则不需打开此地址）
```

### Deploy

```bash
$ npm start
$ npm stop
```
